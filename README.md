# UE5 自动化数值平衡战斗系统

## 项目介绍
- 这是一个用 UE5 C++ 开发的回合制战斗数值平衡项目，目标是把“手工调数值”变成“可自动迭代的算法流程”。
- 项目核心是“仿真引擎 + MCTS 决策 + GA 调参 + 数据驱动配置”，用于验证角色强度、技能收益和战斗节奏是否达到目标。

## 我已经完成的内容
- 用 UE5 C++ 搭建了战斗系统底层框架，核心逻辑与表现层分离，能高频率进行纯逻辑仿真。
- 做了 MCTS 决策模块，让 AI 能在战斗中做前瞻性选择，用于更真实地评估数值强度。
- 做了 GA 自动调参模块，按目标胜率与回合数自动迭代角色属性，减少手工反复试错。
- 建立了数据驱动流程，角色、技能等参数通过 CSV / DataTable 配置，方便批量调参与回归。
- 完成了 CSV 实验结果导出，处理了 Excel 兼容问题（UTF-8 BOM / 分列），可直接看胜率和回合数据。

也就是说目前这个项目可以实现简单的RTB回合制战斗，然后可以导出战斗数据，也支持设置目标胜率与回合通过遗传算法调整数值。
